# Documentation Directory

This directory contains all project documentation, organized by category.

## 📁 Directory Structure

### `/community`
Community guidelines and contribution guides
- **CONTRIBUTING.md** - How to contribute to the project
- **CODE_OF_CONDUCT.md** - Community code of conduct

### `/github-templates`
GitHub issue and PR templates
- **bug_report.md** - Template for bug reports
- **feature_request.md** - Template for feature requests

## 📝 Documentation Standards

- Use clear, descriptive filenames in SCREAMING_SNAKE_CASE
- Include emojis in headers for better visual scanning
- Keep documentation up-to-date with code changes
- Reference the Lemonade Design System for UI/UX decisions
