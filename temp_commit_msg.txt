feat(ui): rebrand to 'Lemon' theme and harden audio service

Major UI overhaul replacing shadcn/ui with a custom 'Lemon' design system
and critical stability fixes for the RealtimeASRService.

UI Changes:
- Introduced 'Lemon' design system in `src/ui/components/lemon/`
  - Added LemonBadge, LemonCard, LemonDialog, LemonSelect, LemonSlider,
    LemonSwitch, LemonTabs
  - Added InterviewCard for consistent list items
- Refactored Dashboard to use new Lemon components
- Updated global styles in `src/ui/index.css` with `--color-lemonade-*` variables
- Removed unused shadcn/ui components (`src/ui/components/ui/*`)
- Removed `components.json` configuration

Audio Service Improvements (`RealtimeASRService.ts`):
- Fixed audio silence issue on Chromium by using native sample rate
  initialization and manual downsampling (48kHz -> 16kHz)
- Implemented 'Circuit Breaker' to prevent infinite loops of empty transcripts
  - Stops capture after 5 consecutive empty/silence responses
  - Emits error when circuit is broken

Other Changes:
- Minor styling updates to VoiceOrb for better contrast in dark mode
